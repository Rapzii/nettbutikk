# Hvordan prokjektet viser kompetanse i faget:

Utvikling: 
Du lager frontend med HTML, CSS, JS og backend med Flask som henter data fra MariaDB.

Drift:
Du setter opp en MariaDB-server, kobler til fra Flask og viser forståelse av databasestyring.

Brukerstøtte: 
Du lager en enkel og ryddig nettside, dokumenterer hvordan systemet brukes, og hjelper brukeren forstå funksjonene.
